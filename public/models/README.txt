Drop your 3D model here as `statue.glb` (binary glTF).

Free starter sources:
  - https://sketchfab.com  (filter: Downloadable + CC license)
  - https://poly.pizza
  - https://opengameart.org

If this file is missing, ARScene falls back to a placeholder shape so the app
still runs end-to-end.

See ../../instructions.md for the full step-by-step.
